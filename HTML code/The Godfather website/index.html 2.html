How to install The Godfather

Overview of the game How to install Operation of the game Controls FAQ

In order to download and play the Godfarther the player has to download visual studio 2020, to download visual studio the user must go to https://visualstudio.microsoft.com/vs/preview/ by copying this link, next the player has to Click the "Download" button for downloading the VS 2019 executable file on the downloaded path. Once this is done Open your system's download path and find the .exe file and then Double-click the .exe file; the Visual Studio installer window will open. Click "Continue, keep in mind the Downloading and installing progress bar window will open after clicking continue. Once here workloads will openand the user then needs to need to select what are the workloads we need. Here, The user needs to select selected ASP.NET and web development, .NET Core cross-platform development, After selecting the workloads, we can see the size of memory for installing the workloads based on our selections and see the installation details in the right side panel. Next the user needs to Click the "Install" button after selecting the workloads. Now, the installation will start and you can see the progress bar for downloading and installing the package. Now that visual sudio is installed the player needs to have a zipped file of the game in their folders, you also need to install monogame, when in visual studio go to extensions and type monogame into the search box. from here the user needs to download the monogame framework and then open the godfarther in moonogame, the last step is to click the consolebutton and then the game is ready to play.
